# Counterevidence

## 1. 该主题已经有两次更短期限盘未兑现

- `Will Russia enter Vasylivka by March 31, 2026?`
- `Will Russia enter Vasylivka by April 30, 2026?`

这意味着：

- 市场过去已经多次把“可能很快突破”的叙事定价过
- 但两次更早期限都没有满足地图判定条件
- 5 月 31 日盘因此不能把“持续攻击”直接等同于“即将进入”

## 2. 决议门槛高于普通战报语言

乌军总参和媒体高频使用：

- `towards Vasylivka`
- `assaults near`
- `repelled attacks`

这些都不是 YES 触发词。

只有以下条件组合才关键：

1. ISW 或 fallback 地图上出现合格着色
2. 着色覆盖 Vasylivka 指定区域任意部分
3. 着色持续一个完整更新周期

## 3. 市场历史价格显示交易者多次上调后又回落

YES 历史价格曾在 4 月下旬和 5 月初冲高，但之后又显著回落到 0.31-0.40 区间。对 MiroFish 来说，这意味着：

- 交易者会对“靠近突破”的叙事快速定价
- 但若没有地图确认，价格可以回吐
- 图谱中应当把 `MARKET_SIGNAL` 与 `OFFICIAL_EVIDENCE` 分开

## 4. 俄军后勤约束是实质性反变量

Reuters 口径的炼油/燃料约束并不直接决定单个村庄战果，但它削弱了持续推进的稳定性。对于这种需要“推进 + 持续 + 地图确认”的盘，后勤约束会放大 NO 路径的概率。

## 5. 截止时间短

从 2026-05-17 到 2026-05-31，剩余时间窗口有限。该盘不是问“最终会不会拿下”，而是问“能不能在很短的窗口内，触发地图规则并持续一个更新周期”。
